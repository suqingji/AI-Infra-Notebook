## 本章简介

选择合适的推理框架是部署高性能推理服务的第一步。本章深入三大主流框架的架构与实战。

**vLLM**是当前最流行的开源推理框架：Scheduler + BlockManager + Worker 架构，支持 PagedAttention、Continuous Batching、Prefix Cache，从模型加载到 OpenAI 兼容 API 的完整部署实战，包括关键配置调优和调度器源码导读。

**SGLang**以 RadixAttention 和 cFSM 结构化输出加速为核心特性，前端提供 SGLang 编程语言模型，特别适合多轮对话、Agent 和结构化输出场景。

**TensorRT-LLM**是 NVIDIA 深度优化的推理方案：Paged KV Cache、Inflight Batching、深度硬件优化，适合追求极限性能且愿意投入适配工作的场景。

**框架选型决策**提供 vLLM（通用快速上手）、SGLang（复杂 Agent）、TensorRT-LLM（极限延迟）的对照表。

**动手实验**：用 vLLM 和 SGLang 分别部署 7B 模型，用相同压测脚本对比 TTFT、TPOT、Throughput。
